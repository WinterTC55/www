# www

The website for this community group.

## Start the website locally

First install Deno.

Mac, Linux:

```sh
curl -fsSL https://deno.land/install.sh | sh
```

Windows:

```sh
iwr https://deno.land/install.ps1 -useb | iex
```

Run the command:

```sh
deno run -A --no-check main.jsx
```

Open http://localhost:8000
